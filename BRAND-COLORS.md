# Brand Colors

These are the primary public-facing color references currently used in SendSwap brand materials and website visuals.

## Core palette

- Ocean Deep: `#031A45`
- Ocean Blue: `#062763`
- Brand Blue: `#0D3989`
- Accent Blue: `#4DABF7`
- Accent Green: `#34D399`
- Accent Gold: `#FBBF24`
- White: `#FFFFFF`
- Soft Text: `#D4E4F7`

## Suggested usage

- Primary backgrounds: `#031A45`, `#062763`, `#0D3989`
- Primary text on dark backgrounds: `#FFFFFF`
- Secondary text on dark backgrounds: `#D4E4F7`
- Accent highlights and interactive emphasis: `#4DABF7`
- Positive or status accents: `#34D399`
- Optional premium highlight: `#FBBF24`

## Note

Brand visuals should stay within the blue ocean-style palette unless there is a strong reason to use an alternate campaign color.

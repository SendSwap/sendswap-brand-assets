# Asset Usage

This file provides basic public guidance for using SendSwap brand assets.

## Brand name

- Use `SendSwap` with capital `S` in both words.
- Do not rewrite the brand as `Send Swap`, `sendswap`, or other variations in titles or headings unless the context requires a domain reference.

## Logo usage

- Use the official logo files without changing the symbol shape.
- Preserve clear spacing around the logo.
- Avoid stretching, rotating, or recoloring the logo in a way that reduces recognition.

## Domain usage

- Prefer `SendSwap` as the brand name in headings and descriptions.
- Use `sendswap.io` when a domain reference is needed.

## Public wording

Suggested neutral description:

`SendSwap is a privacy-first, non-custodial instant crypto exchange for wallet-to-wallet swaps.`

## Not recommended

- unofficial logo edits
- outdated domain references
- inconsistent social handles
- exaggerated claims that are not reflected in public documentation
